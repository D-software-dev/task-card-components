# Task Card Component

A responsive set of task (todo) cards, where individuals can check todo's and mark them as done.

## AIM

The aim of this project is to:

- improve compliance with Web Content Accessibility Guidelines (WCAG).
- train testing practices during development
- improve responsiveness awareness during development

## FEATURES

This project primarily tests accessibility and responsiveness, so its features are tied to this cause, which is not very visual.

## Challenges

- Coming up with a design for the project posed a decent challenge. I ended up using google's stitch (AI UI/UX tool) as my primary inspiration.
- Getting the cards to seamlessly respond to device width automatically (i.e without specific media queries).
- Making UI updates accessible and compatible with Web Content Accessibility Guidelines (WCAG).

## HOW TO ACCESS THE PLATFORM

Clone the repository and open the index.html file with live server. The live link is still pending.
